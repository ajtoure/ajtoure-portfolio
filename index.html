airhead

a meditation app that simulates deep breathing

For this project we made an app that assists the user with their meditation practice. The user selects a time and an animation is generated, along with a quote from our firebase database. The animation runs until the time runs out, at which point the app prompts the user to end their meditation or start again.

  • HTML5
  • CSS3
  • JS6
  • JSX
  • React
  • Firebase
